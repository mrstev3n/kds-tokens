# kds-tokens

## Project setup
```
npm install
```

### Compiles and transform json file for Style dictionary
```
npm run builder
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

or

```
npm run lint -- --fix
```
